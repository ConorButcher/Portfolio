NDA

INTERACTIVE SPACE WITHIN THE METAVERSE

PROJECT DETAILS

CLIENT: NDA

DATE: NOVEMBER 2023

CATEGORY: INTERIOR / EXTERIOR / UE5 / METAVERSE

Welcome to my portfolio, where I delve into the intricate process of crafting a 3D space destined for the metaverse. This project is a testament to my skills in Autodesk 3DS Max and Unreal Engine, showcasing a seamless workflow that brings virtual environments to life. Let's explore the key elements that make this virtual space a captivating and interactive experience.

Autodesk 3DS Max and Unreal Engine 5 Workflow

While working between Autodesk 3DS Max and Unreal Engine, i made use of the plugin called Datasmith, allowing a smooth transition of assets. This streamlined workflow allowed for efficient creation, modification, and export of 3D models, maintaining the properties adjusted within UE like UV Maps for example.

Lumen Lighting System

Leveraging Unreal Engine 5's Lumen lighting system, I adjusted various settings to craft realistic and dynamic lighting feel. Lumen's global illumination capabilities can be fairly taxing on hardware, so many optimisations were required.

Volumetric and Interactive Volumetric Fog

The incorporation of volumetric and interactive volumetric fog added depth and atmosphere to the environment. Users can interact with the fog in certain areas, creating a dynamic and ever-changing ambiance that adds to the immersive experience.

Texturing Within Unreal Engine 5

Unreal Engine 5's powerful texturing capabilities allowed me to create stunning, high-fidelity textures directly within the engine. This not only improved efficiency but also facilitated essential adjustments for optimal performance.

Blueprint Design for Interactive and Working Elevators

Through blueprint design, I implemented fully functional elevators within the virtual space. Users can seamlessly interact with and navigate these elevators, enhancing the overall interactivity of the environment.

Sound Design

Sound design played a crucial role in creating an immersive experience. Carefully chosen audio tracks, from ambient music for seperate rooms to interactive elevator mechanisms. I also made use of effects to add echo to the audio tracks which play in large spaces.

Animated LED Screens Using PNG Sequences

Animated LED screens brought life and movement to the environment. Leveraging PNG sequences, I created captivating visuals that added a dynamic and futuristic touch to the virtual space. For some extra detail I added an effect where you can see the idividual pixels when upclose to the screens, i removed this however to improve performance.

Various Optimization Methods

To ensure a seamless experience across various platforms, I implemented optimization techniques. These included LOD (Level of Detail) adjustments for texture maps, texture compression, and efficient asset management to maintain performance without compromising visual quality. The GPU Profiler, CPU Profiler and Statistic tools help to located unoptimised assets.

Packaging for Testing on PC Platforms

The project was packaged for testing on PC platforms. This allows for clients to run the game without having to have Unreal Engine installed on their machine.

Rigged Characters and Nanite

Rigged characters were incorporated into the virtual space, bringing it to life with animated inhabitants. Utilizing Unreal Engine 5's Nanite technology, I ensured that high-polygon characters could be seamlessly integrated, maintaining visual fidelity.

In crafting this 3D space for the metaverse, I've demonstrated a commitment to pushing the boundaries of virtual world creation. Each element, from the technical aspects to the creative design, contributes to an immersive and captivating user experience. This project serves as a testament to my skills in 3D modeling, UE5 development, and the creation of visually stunning and interactive environments within the metaverse.
